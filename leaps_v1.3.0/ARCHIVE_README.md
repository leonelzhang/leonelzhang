## 项目概述
Leaps是一个功能强大的纳斯达克股票价格可视化系统，提供图形化界面用于查看和分析股票价格走势，集成多种技术指标和智能买卖信号。

## 核心功能

### 1. 股票数据获取
- 支持纳斯达克股票实时数据获取
- 多种时间周期选择（1mo到max）
- 多种数据间隔（1d, 5d, 1wk, 1mo, 3mo）
- 基于yfinance库的数据源

### 2. 图表可视化
- **折线图**: 显示价格走势
- **K线图**: 显示开盘价、最高价、最低价、收盘价
- **成交量图**: 显示交易量

### 3. 技术指标
- **移动平均线**: MA20、MA50、MA200
- **布林带**: 基于标准差的波动率指标
- **MACD指标**: 移动平均收敛发散指标
  - MACD线（EMA12 - EMA26）
  - 信号线（MACD的9日EMA）
  - 柱状图（MACD - 信号线）

### 4. 智能买卖信号
- **买入信号 (B)**: 
  - 价格突破近期最高价
  - MACD > 0（新增过滤条件）
  - 绿色▲标记
- **卖出信号 (S)**:
  - 价格跌破近期最低价
  - MACD < 0（新增过滤条件）
  - 红色▼标记
- **信号周期**: 1mo、3mo、6mo、1y

### 5. 数据导出
- 支持导出为CSV格式
- 支持导出为Excel格式
- 图表保存为PNG/PDF格式

## v1.3.0 新增功能

### MACD过滤机制
- ✨ 买入信号仅在MACD > 0时显示
- ✨ 卖出信号仅在MACD < 0时显示

- 📊 显示MACD线、信号线、柱状图

## 信号说明

### 买入信号 (B)
- **条件1**: 价格突破近期最高价
- **条件2**: MACD > 0
- **含义**: 价格创新高且处于上升趋势
- **可靠性**: ⭐⭐⭐⭐

### 卖出信号 (S)
- **条件1**: 价格跌破近期最低价
- **条件2**: MACD < 0
- **含义**: 价格创新低且处于下降趋势
- **可靠性**: ⭐⭐⭐⭐
